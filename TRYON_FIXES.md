# 🔧 Исправления TryOn - Камера и Фото

## ✅ Что было исправлено

### 1. **Камера (черный экран)**
**Проблема**: Видео не инициализировалось правильно
**Решение**:
- Добавлена задержка 100ms перед установкой srcObject (для инициализации элемента)
- Добавлена проверка `video.play()` с обработкой ошибок
- Добавлены лучшие constraints (1280x720)
- Добавлен `muted` атрибут на видео элемент
- Добавлены обработчики ошибок `onError` и `onLoadedMetadata`

### 2. **Видео CSS**
**Проблема**: Видео могло не отображаться
**Решение**:
- Явно установлены размеры: `width: 100%` и `height: 400px`
- Добавлена трансформация `transform: scaleX(-1)` для зеркального отражения (как в камере)
- Изменено с `max-height` на фиксированную `height`
- Добавлено `display: block` для правильной разметки

### 3. **Захват фото с камеры**
**Проблема**: Canvas был пуст, blob мог быть null
**Решение**:
- Добавлена проверка `readyState !== HAVE_ENOUGH_DATA`
- Добавлена конвертация blob → File для консистентности
- Улучшена обработка ошибок с информативными сообщениями
- Добавлены логи для дебага

### 4. **Загрузка на сервер**
**Проблема**: 401 Unauthorized, 404 Not Found
**Решение**:
- Добавлена проверка `product.id` перед отправкой
- Улучшена обработка ошибок API (парсинг JSON, fallback)
- Добавлены логи с деталями запроса
- Добавлена проверка `result.image` перед использованием

### 5. **Обработка ошибок камеры**
**Добавлены специфичные сообщения для**:
- `NotAllowedError` - "Доступ к камере запрещен"
- `NotFoundError` - "Камера не найдена"
- `NotReadableError` - "Камера занята другим приложением"

## 📝 Инструкция по тестированию

### Вариант 1: Загрузка фото
1. Откройте `http://localhost:5174/products`
2. Нажмите "🎨 Try On" на товаре
3. Выберите "📤 Загрузить фото"
4. Загрузите фото (должно быть видимо лицо и тело)
5. Выберите тип одежды
6. Нажмите "🎨 Примерить"
7. Должно появиться изображение результата

### Вариант 2: Камера
1. Откройте modal Try On
2. Нажмите "📷 Камера"
3. Нажмите "🎥 Включить камеру"
4. Подождите загрузки видео (черный -> с видео)
5. Позиционируйте себя в кадре
6. Нажмите "📸 Сделать фото"
7. Должно переключиться на режим загрузки с вашей фотографией
8. Выберите тип одежды и нажмите "🎨 Примерить"

## 🐛 Отладка (в консоли браузера)

Откройте DevTools (F12) → Console

**Вы должны увидеть логи:**
```
Video loaded, dimensions: 1280 x 720
Sending try-on request with: {product_id: 7, clothing_type: "top", file_size: 145000}
Response status: 200
Result: {success: true, image: "iVBORw0K...", ...}
```

**При ошибке вы увидите:**
```
Camera error: NotAllowedError
Upload error: Ошибка 401: Unauthorized
```

## ⚠️ Частые проблемы

| Проблема | Решение |
|----------|---------|
| Камера черный экран | Проверьте разрешение браузера на камеру (Safari → Preferences) |
| "Доступ запрещен" | Откройте настройки приватности и включите камеру для браузера |
| 401 Unauthorized | Разлогиньтесь и заново залогиньтесь |
| 404 Not Found | Проверьте что товар существует в БД |
| "Ошибка при обработке" | Загрузите фото где видна фигура (от головы до ног) |
